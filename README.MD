# Open-Source GDPS Launcher [ OSGDPSL ]
(look at: https://osgdpsl.141412.xyz/2025/11/14/open-source-gdps-launcher-osgdpsl/)

Hello! OSGDPSL is a GDPS project designed to make developing a GDPS launcher easier.

## Features  [ FEACHER ]

    Supports wallpapers!
    As of now, 5 different styles!
    You can add your own style via CSS!
    Has documentation for every feature!
    Very customizable!

## Features [ INSIDE LAUNCHER ]

    * Unlike DGDPSL, this version replaces “gd22” with “$(Your GDPS Name) 2.2”!
    * Sleek UI!
    * Dark & Light mode!
    * Simple, yet easy to use!

## FAQ [FREQUENTLY ASKED QUESTIONS]

Q: Can’t I just use MegaSa1nt’s launcher?

    A: Yes, absolutely! Still, I won’t really recommend it since it’s only made for GCS.

Q: Why would I need a GDPS launcher?

    A: As a GDPS owner, I would say that’s because it makes your GDPS unique (Unless everyone starts using it..)

Q: How did you make this launcher?

    A: Very simple, I used EdgeWebView, HTML, JavaScript, cURL and CSS!
